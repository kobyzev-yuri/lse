Анализатор эффективности сделок

Единый расчёт с Telegram командой /analyser. Период 1-30 дней. Ниже — человекочитаемая выжимка; полный отчёт всегда в JSON с того же API (удобно сохранить или отдать скрипту).

Результирующий JSON (локально, скрипты, jq)

Один запрос GET /api/analyzer возвращает весь отчёт: meta, summary, top_cases, llm, auto_config_override и при необходимости trade_effects. Параметры формы выше (дни, стратегия, LLM) подставляются в ссылки и в скачивание.

Открыть в браузере (можно «Сохранить как…»):

  • Полный отчёт — как после «Запустить анализ»
  • То же + массив trade_effects по каждой сделке

Путь для curl / копирования:

curl -sS "https://ВАШ_ХОСТ/api/analyzer?days=7&strategy=GAME_5M&use_llm=0" -o report.json
