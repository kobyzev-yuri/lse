Исполнение торгового цикла

Портфельная игра по дневным стратегиям: для каждого тикера запрашивается сигнал (AnalystAgent + LLM). При BUY/STRONG_BUY открывается позиция (если её ещё нет), затем проверяются стоп-лоссы по всем открытым позициям. То же самое делает крон trading_cycle_cron.py по расписанию (9:00, 13:00, 17:00). Запуск вручную — чтобы «прогнать» цикл сейчас по выбранным тикерам.
