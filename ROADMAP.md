## 🔮 Приоритет 3: Продвинутые функции

### Улучшение технических индикаторов (2-3 часа)

**EMA вместо SMA для волатильных активов:**
- Добавить расчет EMA_5 в дополнение к SMA_5
- Использовать EMA для волатильных активов (SNDK) - быстрее реагирует на изменения
- Использовать SMA для стабильных активов (MSFT) - меньше ложных сигналов
- Статус: ⏳ В планах

### Multi-Asset Rebalancing (3-4 часа)

- Автоматическая ребалансировка портфеля
- Ограничение доли одного актива (например, 20%)
- Частичное закрытие позиций при превышении лимита

### Векторная база знаний (4-6 часов)

- Семантический поиск похожих исторических событий
- Интеграция в процесс принятия решений
- Анализ исходов похожих ситуаций

### Real-time API Integration (6-8 часов)

- Интеграция с реальными брокерскими API
- Демо-счет для тестирования
- Управление рисками в реальном времени
